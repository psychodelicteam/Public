# CONTRIBUTING

Obrigado por querer contribuir com o Psychodelic Team!

## Processo rápido
1. Faça um fork do repositório.
2. Crie uma branch: `git checkout -b feat/sua-contrib`
3. Faça commits claros e testes onde aplicável.
4. Abra um Pull Request com descrição do que foi feito.

## Onde colocar
- Ferramentas → `/tools/<seu-projeto>`
- Guias → `/guides/<seu-guia>`
- Recursos → `/resources/`
- Casos públicos (OSINT) → `/cases-publicos/`
- Submissões da comunidade → `/community/` (coloque link pro seu repo e README)

## Requisitos mínimos para PR
- README explicando o projeto
- Instruções de uso
- Nota atestando que o conteúdo usa apenas dados públicos e legais
- LICENSE ou indicação de licença (recomenda-se MIT/Apache2/GPLv3)

## Código de conduta
Siga o `CODE_OF_CONDUCT.md`.
